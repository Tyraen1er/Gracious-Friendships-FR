# Gracious-Friendships-FR
Traduction pour le jeu WOTR du mod - Gracious Friendship

# Installation
Copier le fichier frFR.json et le coller dans le dossier du Localization du mod.
Normalement accessible ici pour Windows : %LocalAppData%Low\Owlcat Games\Pathfinder Wrath Of The Righteous\Modifications\GraciousFriendships\Localization
